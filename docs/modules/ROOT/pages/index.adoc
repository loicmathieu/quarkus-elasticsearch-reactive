== Installation

If you want to use this extension, you need to add the `io.quarkiverse.quarkus-elasticsearch-reactive:quarkus-elasticsearch-reactive` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.quarkus-elasticsearch-reactive</groupId>
    <artifactId>quarkus-elasticsearch-reactive</artifactId>
</dependency>
----
